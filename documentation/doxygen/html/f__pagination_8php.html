Retrieve paginated data using a fetch function and a total count function.

Parameters
mysqli$connThe database connection object.
callable$fetchFunctionA callback function to fetch data for the current page. It should accept $conn, $offset, and $recordsPerPage as arguments.
callable$totalFunctionA callback function to get the total number of records. It should accept $conn as an argument.
int$recordsPerPageThe number of records to display per page.
Returns
array An associative array containing:
  • 'items': The fetched items for the current page.
  • 'totalPages': The total number of pages.
  • 'currentPage': The current page number.
